MT5 Ribbon V5.0 - Aggressive Monitor Pro 🚀🔥

تيرمينال تداول عالي الأداء ومتعدد المسارات (Multi-threaded) لمنصة MetaTrader 5، مبرمج بلغة بايثون. مصمم خصيصاً للمتداولين بنظام السكالبينج الذين يحتاجون إلى سرعة تنفيذ فائقة ومراقبة لحظية للصفقات.

🖥️ عرض واجهة البرنامج


تصميم عصري يركز على سرعة التنفيذ ومراقبة الأرباح اللحظية.

✨ الميزات الرئيسية

🚀 التنفيذ الهجومي (Aggressive Execution): محرك تنفيذ مخصص مع نسبة سماح عالية للانحراف السعري (Deviation) لتجاوز أخطاء "لا يوجد أسعار" (No Prices) وأخطاء إعادة التسعير.

🧵 التنفيذ المتعدد السريع: فتح صفقات متعددة في نفس اللحظة دون تجميد واجهة المستخدم بفضل تقنية المسارات المتعددة.

📊 مراقب الصفقات المباشر: لوحة مخصصة لعرض التذاكر المفتوحة، أحجام العقود، والأرباح والخسائر اللحظية لكل صفقة.

🛡️ تأمين الأرباح الذكي (Break-Even): تأمين صفقاتك الرابحة تلقائياً عن طريق نقل وقف الخسارة إلى سعر الدخول.

⚡ أولوية قصوى للنظام: استخدام Windows API لرفع أولوية المعالجة وضمان أقل زمن تأخير (Latency).

📈 مراقبة الصفقات النشطة


تتبع كل تذكرة (Ticket) على حدة مع تحديث مباشر للحالة والربح.

⚠️ خطوة هامة: تفعيل التداول الآلي في MT5

لكي يتمكن البرنامج من إرسال الأوامر بنجاح، يجب تفعيل الخيارات التالية داخل منصة MetaTrader 5:

افتح منصة MetaTrader 5.

اضغط Ctrl+O لفتح قائمة Options.

انتقل إلى تبويب Expert Advisors.

قم بتحديد الخيارات التالية:

✅ Allow Algo Trading

✅ Allow DLL imports

اضغط OK.

🚀 كيفية التشغيل

لديك خياران لتشغيل التيرمينال:

1️⃣ الخيار الأول: نسخة الـ EXE (للمتداولين)

افتح مجلد MT5-Ribbon.

قم بتشغيل ملف MT5-Ribbon.exe كمسؤول (Run as Administrator).

2️⃣ الخيار الثاني: الكود المصدري (للمطورين)

تأكد من تثبيت المكتبات:

pip install MetaTrader5 customtkinter pywin32


شغل السكربت:

python MT5-Ribbon.py
